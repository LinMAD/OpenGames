#### Soviet tetris with SDL2

`Status`: In development...

![](assets/v01.png)

Project:
```text
├── main
│   └── src
│       ├── GameHandler  // Contains game loop (input, update, draw)
│       ├── Component    // Game components  
│       └── Setting      // General game enums settings
└────────── Util         // Utilities, like random generator
```
